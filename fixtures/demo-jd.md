## 公司

某新锐 AI SaaS 公司（B 轮，团队 60 人），主攻面向企业的智能客服与知识检索产品。

## 岗位职责

1. 负责公司核心 LLM 应用后端服务的设计与开发，包括对话编排、检索增强（RAG）、工具调用等关键链路
2. 主导高并发场景下的系统稳定性建设，参与设计支撑日均千万级请求的服务架构
3. 与算法团队协作，将研究成果工程化落地，把模型能力变成可上线的产品功能
4. 带领 2-3 人小组完成模块级开发任务，对模块质量、进度、技术债务负责
5. 参与技术选型决策，对线上事故有 ownership，参与值班 oncall

## 任职要求

**硬技能（必备）**
- 3 年以上后端开发经验，精通 Python（FastAPI / Django 任一）
- 熟悉分布式系统设计，理解 CAP、一致性、最终一致性、分布式锁等概念
- 熟悉至少一种主流消息队列（Kafka / RabbitMQ / Pulsar）
- 熟悉 Kubernetes，能独立完成服务部署、监控、问题排查
- 熟悉 Postgres 或 MySQL，理解索引设计、查询优化、读写分离
- 有 LLM 应用（RAG / Agent / Function Calling）相关的工程化经验

**软技能与加分项**
- 有带 2-3 人小组的经验（"leader experience"）
- 有从 0 到 1 设计并上线核心服务的完整经历
- 能在跨职能团队（产品/算法/前端）中有效沟通，推动事项落地
- 加分：开源项目贡献、技术博客、相关领域论文
- 加分：熟悉向量数据库（Qdrant / Milvus / pgvector）
- 加分：熟悉 Anthropic / OpenAI / 国产大模型 API 的工程实践

## 我们提供

- 有竞争力的薪酬 + 期权
- 灵活办公（每周 2 天 WFH）
- 直接接触公司最核心的技术决策
